Preset Market Seed

Experimental admin-only market tool based on IceHunter's MIT-licensed marketbot research, with attribution in the README.

Seeds the exchange with NPC listings owned by the {{ market_bot_class }} bot actor: 2 of each equippable item, 2 of each schematic, and {{ market_resource_stack_size }} of each resource-like stackable.

Vehicle mobility parts with names or IDs containing wing, track, or locomotion are boosted to 8 listings.

Refined resources are priced 2.5x higher than the normal resource baseline.

Raw resources are priced 5x higher, with targeted exceptions: Spice Sand/Residue and Agave Seeds 10x, Titanium Ore and Stravidium Mass 8x, Basalt Stone 0.2x.

Prices use the multiplier below, defaulting to {{ market_price_multiplier }}x, so Solari keeps value. The clear controls affect only this bot's NPC listings; they do not remove player listings.

{{ market_bot_class }} can also buy player listings priced at or below the configured percent of what this preset would list the same item for. Automated buyback sweeps run once immediately when started, then use the interval below while Easy Dune Admin is running.

Vehicle Module Repair

Experimental admin-only tool.

This updates CurrentDurability and MaxDurability on every module in dune.vehicle_modules for the selected vehicle. It is intended for repairing vehicle modules, not cloning or spawning vehicles.

Recommended first use: test on a low-risk vehicle and confirm in-game behavior after relog/reload.

Admin Gifts & Progression

Admin-only helper tools.

Lasgun kit and Solaris grants use RedBlink's normal item grant command. Research points directly edit the selected character's progression JSON, so the character should be offline before changing points.
Grants: Experimental Lightning Gun, Lasgun Mod, Accuracy Mod, and Damage Mod.
Character XP controls the actual displayed character level and is separate from specialization XP. It caps at {{ "{:,}".format(character_max_xp) }} XP / level 200 and recalculates skill points plus research points from the new level.
Set Character Level uses the same level XP curve as Character XP, then recalculates skill points and research points. The character should be offline before changing level.
WIP / unconfirmed: Specialization XP is separate from character level XP. This tool appears to create or update the expected database entries, but persistence and in-game behavior are not yet verified because current testing has not reached the required progression/faction access. Each specialization track appears capped at {{ "{:,}".format(specialization_max_xp) }} XP; stay at or below that per track while testing. After progression edits, a map restart or full battlegroup restart may be required; restarts can appear slow, and login may briefly show an error before recovering.
Reset Specialization deletes specialization XP rows. Selecting All also clears purchased specialization keystones, which is useful after testing but may require relogging, a map restart, or a full battlegroup restart before the UI catches up.
Experimental progression presets edit journey story-node completion state and should be used only while the player is offline. In normal play, journey nodes may also apply tags/flags, reveal quests or lore, advance tutorials, alter faction tier/reputation, or gate rewards and follow-up systems. This tool updates the journey rows we can safely target, but it may not reproduce every side effect from live gameplay. Relog after applying or resetting. A map restart or full battlegroup restart may be required; restarts can appear to take a long time, and login may briefly show an error before recovering.

Vehicle Teleport

Experimental admin-only tool.

Map restart required: vehicle actors already loaded by the server keep their old in-memory location until the affected map/server instance is restarted.

Ownership is not confirmed in the current schema, so this tool can move any detected supported vehicle actor. Use a database backup and test with a low-risk vehicle first.

This updates dune.actors.map, partition_id, and XYZ position while preserving the current rotation.

Aim high on Z. A value below terrain can place the vehicle underground, so overestimate altitude first and lower it only after confirming in-game.
